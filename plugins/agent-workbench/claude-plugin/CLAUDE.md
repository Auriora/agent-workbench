# Agent Workbench Claude Code Guidance

When a task needs repository investigation, change evidence, or validation
planning and the Agent Workbench MCP server is available, invoke the packaged
`/agent-workbench:agent-workbench` skill. Skip this for trivial tasks.

The skill teaches the provider-neutral MCP workflow; it does not invoke the
runtime automatically or add Claude-specific runtime behavior.
